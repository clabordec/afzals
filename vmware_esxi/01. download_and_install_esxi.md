<h1>Download and Install VMWare ESXi</h1>
This project outlines downloading and installing VMWare ESXi within VMWare Workstation Player.<br />

<br />

<h2>Environments and Technologies Used</h2>

- VMWare Workstation Player
- VMWare ESXi


<h2>Operating Systems Used</h2>

- Windows 11 Home


<h2>High-Level Deployment and Configuration Steps</h2>

- Download and install the ESXi server onto a virtual machine within VMWare Workstation Player to create multiple and different virtual machine running multiple operating systems


<br>
